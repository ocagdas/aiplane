# aiplane v0.1.0 — Developer preview

This first developer preview introduces the narrow public workflow: describe a local or hybrid AI development environment in a reviewable profile, inspect it, diagnose readiness, recommend suitable model configuration, and export deterministic configuration for existing tools.

## Primary workflow

```bash
aiplane quickstart local-coding --dry-run
aiplane discover
aiplane doctor
aiplane recommend
aiplane export continue
```

Exports print configuration for review; they do not install or edit the target tool.

## Supported surface

Portable profile bootstrap/validation/rendering, discovery coverage reporting, doctor, recommendation, policy explanation, and Tier-1 exports are tested from installed wheels on Linux, macOS, and Windows.

Tier-1 export contracts are Continue, Aider, generic OpenAI-compatible, and generic MCP. Other commands remain advanced or experimental and do not expand the developer-preview promise.

Runtime installation/update helpers are supported only on native Ubuntu/Debian. Windows supports SSH tunnel planning but not helper-managed tunnel lifecycle. See [platform support](../../user/platform-support.md).

## Install

Download the wheel and `SHA256SUMS` from this release, verify the checksum, then choose one installation owner:

```bash
uv tool install ./aiplane-0.1.0-py3-none-any.whl
pipx install ./aiplane-0.1.0-py3-none-any.whl
python -m pip install ./aiplane-0.1.0-py3-none-any.whl
```

Package-index installation is not advertised in this preview.

## Integrity

Release automation generates `SHA256SUMS` from the wheel and source distribution after building them and verifies the manifest before publication. Signing is deferred until a maintainable trusted signing identity is configured; no ad-hoc signature is claimed.

## Upgrade, uninstall, and rollback

Follow the exact commands in [setup](../../user/setup.md#standard-wheel-install-no-repository-clone). Preserve reviewed profile YAML before changing versions. To roll back, uninstall with the same tool that installed Aiplane and reinstall the previously downloaded immutable wheel after verifying its checksum. Published versions are never retagged or silently replaced.

## Developer-preview limitations

This release is pre-1.0 alpha software. External-user demonstrations and the final post-trial documentation sweep remain readiness work. Profiles are not signed, redaction is best effort, and Aiplane is not a secret manager, runtime, model proxy, coding agent, or hidden cloud deployment engine.
